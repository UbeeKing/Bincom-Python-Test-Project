## Bincom ICT Python Developer Test

A solution to the Bincom ICT Python developer screening test.

### Features
- Web scraping with BeautifulSoup to extract color data from HTML
- Statistical analysis — mean, median, variance and probability
- Recursive binary search algorithm
- Binary to base 10 conversion
- Fibonacci sequence sum

### Technologies Used
- Python 3.14
- BeautifulSoup4
- Collections
- Statistics

### How To Run
1. Clone the repository
2. Install dependencies: pip install beautifulsoup4
3. Run: python Bincom.py
